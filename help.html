Focus Nowâ€™s goal is to provide a place for users to easily manage their work with a customizable timer. The intention of this app is to help users maintain a balance between their focus and break time in order to avoid constant distractions while they study, code, etc.
How to use Focus Now timer?
  1. Add task to be focused on.
  2. Select task to work on in calendar.
  3. Start timer (and focus on the task for the set time.)
  4. Take a break (5 minutes recommended.)
  5. Repeat (until all tasks are completed.)
Features
  • - Create task allows the user to set up the task they want to stay focused on while the timers run.
  • - Calendar shows the user where their tasks are scheduled for the day.
  • - Activity History lets the user reflect on how long they successfully focused on their tasks over the week.
